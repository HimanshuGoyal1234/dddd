Atomic Structure Formulas

Basic Atomic Properties

Atomic Number (Z):
Z = number of protons = number of electrons (in neutral atom)
Mass Number (A):
A = number of protons (Z) + number of neutrons (N)
Nuclear Charge:
Nuclear charge = +Ze
where e = elementary charge (1.602 × 10⁻¹⁹ C)

Bohr's Model Formulas

Radius of nth orbit:
rₙ = (n²h²ε₀)/(πmZe²) ≈ 0.529 × n²/Z Å
n = principal quantum number
h = Planck's constant (6.626 × 10⁻³⁴ Js)
ε₀ = permittivity of free space
m = mass of electron (9.109 × 10⁻³¹ kg)
Z = atomic number
1 Å (angstrom) = 10⁻¹⁰ m
Velocity of electron in nth orbit:
vₙ = (Ze²)/(2ε₀nh) ≈ 2.18 × 10⁶ Z/n m/s
Energy of electron in nth orbit:
Eₙ = - (mZ²e⁴)/(8ε₀²n²h²) ≈ -13.6 Z²/n² eV
1 eV = 1.602 × 10⁻¹⁹ J

Quantum Mechanical Model

De Broglie Wavelength:
λ = h/p = h/(mv)
p = momentum
m = mass of particle
v = velocity of particle
Heisenberg Uncertainty Principle:
Δx × Δp ≥ h/(4π)
Δx = uncertainty in position
Δp = uncertainty in momentum
Schrödinger Wave Equation:
Ĥψ = Eψ
Ĥ = Hamiltonian operator
ψ = wave function
E = energy of the system

Electron Configuration Rules

Maximum number of electrons in a shell:
2n²
n = principal quantum number
Number of orbitals in a subshell:
2l + 1
l = azimuthal quantum number
Maximum number of electrons in a subshell:
2(2l + 1)

Photoelectric Effect

Einstein's Photoelectric Equation:
hν = hν₀ + ½mv²
ν = frequency of incident light
ν₀ = threshold frequency
m = mass of electron
v = velocity of ejected electron
Work Function:
Φ = hν₀
Kinetic Energy of Ejected Electron:
KE = hν - Φ

X-ray Spectra

Moseley's Law:
√ν = a(Z - b)
ν = frequency of X-ray
a, b = constants
Z = atomic number
Bragg's Law:
nλ = 2d sinθ
n = order of reflection
λ = wavelength
d = interplanar distance
θ = glancing angle

Radioactive Decay

Decay Law:
N = N₀e^(-λt)
N = remaining quantity
N₀ = initial quantity
λ = decay constant
t = time
Half-Life:
t₁/₂ = ln(2)/λ ≈ 0.693/λ
Average Life:
τ = 1/λ
Activity (Radioactivity):
A = -dN/dt = λN
