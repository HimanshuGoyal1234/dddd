Quantum Numbers Basics

Four quantum numbers describe each electron in an atom:

Number Symbol Description
Principal n Energy level/shell (1, 2, 3...)
Azimuthal l Subshell shape (0 to n-1)
Magnetic m Orbital orientation (-l to +l)
Spin s Electron spin (±½)
Azimuthal Quantum Number
l = 0 to (n - 1)

Determines the shape of the orbital:

l Value Subshell Shape
0 s Spherical
1 p Dumbbell
2 d Cloverleaf
3 f Complex
Magnetic Quantum Number
m = -l to +l (integer values)

Number of orbitals per subshell = 2l + 1

Example

For p orbitals (l=1):

m = -1, 0, +1 → 3 orbitals

Electrons per Shell
Max electrons = 2n²

Where n is the principal quantum number

Examples

n=1 (K shell): 2(1)² = 2 e⁻

n=2 (L shell): 2(2)² = 8 e⁻

n=3 (M shell): 2(3)² = 18 e⁻

Orbitals per Subshell
Number of orbitals = 2l + 1
Subshell l Value Orbitals
s 0 1
p 1 3
d 2 5
f 3 7
Electrons per Subshell
Max electrons = 2(2l + 1)

Each orbital holds 2 electrons

Subshell Max Electrons
s 2
p 6

Spin Quantum Number
s = +½ or -½

Pauli Exclusion Principle:

No two electrons in an atom can have identical quantum numbers.

Implication

Each orbital can hold maximum 2 electrons with opposite spins.

Subshell Energy Order
(n + l) Rule

Lower (n + l) values fill first. If equal, lower n fills first.

Order: 1s < 2s < 2p < 3s < 3p < 4s < 3d < 4p < 5s < 4d < 5p < 6s < 4f < 5d < 6p < 7s

Example

4s (4+0=4) fills before 3d (3+2=5)
