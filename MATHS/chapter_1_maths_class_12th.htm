Chapter 1: Relations and Functions

Introduction to Relations and Functions

This chapter introduces fundamental concepts of relations and functions which form the building blocks for higher mathematics. You'll learn about different types of relations, functions, their composition, and invertible functions. Understanding these concepts is crucial for calculus and other advanced mathematical topics.

Relation

A relation R from a set A to a set B is a subset of the Cartesian product A × B. The subset is derived by describing a relationship between the first element and the second element of the ordered pairs in A × B.

R ⊆ A × B

Function

A function f from a set A to a set B is a special type of relation where every element of set A has one and only one image in set B.

f: A → B

Functions are also called mappings or transformations.

Types of Functions

1. One-One Function (Injective Function)

A function f: X → Y is called one-one (or injective) if distinct elements of X have distinct images in Y.

f(x₁) = f(x₂) ⇒ x₁ = x₂ for all x₁, x₂ ∈ X

Example: f: R → R defined by f(x) = 3x + 5 is one-one because if f(x₁) = f(x₂), then 3x₁ + 5 = 3x₂ + 5 ⇒ x₁ = x₂.

Horizontal Line Test: A function is one-one if no horizontal line intersects its graph more than once.

2. Onto Function (Surjective Function)

A function f: X → Y is called onto (or surjective) if every element of Y is the image of some element of X under f.

For every y ∈ Y, there exists an x ∈ X such that f(x) = y

Example: f: R → R defined by f(x) = x³ is onto because for any real number y, there exists a real number x = y1/3 such that f(x) = y.

3. Bijective Function

A function f: X → Y is called bijective if it is both one-one and onto.

Example: f: R → R defined by f(x) = 2x + 3 is bijective because it is both injective and surjective.

Key Points:

  • A bijective function has an inverse function
  • The composition of two bijective functions is bijective
  • For finite sets X and Y, |X| = |Y| if a bijection exists
  • Bijective functions are also called one-to-one correspondences

4. Identity Function

The identity function on a set A is the function idₐ: A → A defined by idₐ(x) = x for all x ∈ A.

Example: idₐ: R → R defined by idₐ(x) = x is the identity function on R.

Property: For any function f: A → B, we have f ∘ idₐ = f and idʙ ∘ f = f.

5. Constant Function

A function f: X → Y is called constant if there exists an element c ∈ Y such that f(x) = c for all x ∈ X.

Example: f: R → R defined by f(x) = 5 for all x ∈ R is a constant function.

Composition of Functions

Let f: A → B and g: B → C be two functions. Then the composition of g and f, denoted by g∘f, is the function from A to C defined by (g∘f)(x) = g(f(x)) for all x ∈ A.

Example: Let f: R → R be defined by f(x) = x² and g: R → R be defined by g(x) = cos x. Then (g∘f)(x) = g(f(x)) = cos(x²).

Properties of Composition:

  • Associative: h∘(g∘f) = (h∘g)∘f
  • Composition of two one-one functions is one-one
  • Composition of two onto functions is onto
  • f∘idₐ = f = idʙ∘f (Identity property)
  • Composition is generally not commutative (f∘g ≠ g∘f in most cases)

Theorem:

If f: A → B and g: B → C are both bijective functions, then g∘f: A → C is also bijective.

Invertible Functions

A function f: X → Y is invertible if there exists a function g: Y → X such that g∘f = idₓ and f∘g = idʏ. The function g is called the inverse of f and is denoted by f⁻¹.

Example: The function f: R → R defined by f(x) = 2x + 3 is invertible with inverse f⁻¹(y) = (y - 3)/2.

Key Results:

  • A function is invertible if and only if it is bijective
  • If f and g are invertible, then (g∘f)⁻¹ = f⁻¹∘g⁻¹
  • The inverse of a function is unique
  • Graph of f⁻¹ is the reflection of graph of f about the line y = x

Note: Only bijective functions have inverses. For a function f: A → B:

  • If f is injective but not surjective, we can define an inverse from f(A) to A
  • If f is surjective but not injective, we can define a right inverse (but not a unique one)

Binary Operations

A binary operation ∗ on a set A is a function ∗: A × A → A. We denote ∗(a, b) by a ∗ b.

Examples:

  • Addition (+) is a binary operation on R
  • Multiplication (×) is a binary operation on R
  • Union (∪) is a binary operation on the power set of a set
  • Matrix multiplication is a binary operation on the set of n×n matrices

Properties of Binary Operations:

  • Commutative: a ∗ b = b ∗ a for all a, b ∈ A
  • Associative: (a ∗ b) ∗ c = a ∗ (b ∗ c) for all a, b, c ∈ A
  • Identity element: An element e ∈ A such that a ∗ e = e ∗ a = a for all a ∈ A
  • Inverse: For each a ∈ A, there exists b ∈ A such that a ∗ b = b ∗ a = e
  • Distributive: a ∗ (b • c) = (a ∗ b) • (a ∗ c) for another operation •

Theorem:

The identity element of a binary operation, if it exists, is unique. Similarly, the inverse of an element, if it exists, is unique.

Practice Problems

20 Practice Questions

1. Show that the function f: R → R defined by f(x) = 4x + 3 is invertible and find its inverse.

2. Let A = {1, 2, 3}, B = {4, 5, 6, 7} and f = {(1,4), (2,5), (3,6)} be a function from A to B. Show that f is one-one but not onto.

3. Consider the binary operation ∧ on the set {1, 2, 3, 4, 5} defined by a ∧ b = min{a, b}. Write the operation table for this operation.

4. Let f: N → N be defined by f(n) = n + 1 if n is odd and f(n) = n - 1 if n is even. Show that f is bijective.

5. Let f: R → R be defined as f(x) = x⁴. Is f one-one? Is f onto? Justify your answer.

6. Let A = R - {3} and B = R - {1}. Consider the function f: A → B defined by f(x) = (x-2)/(x-3). Show that f is bijective.

7. Let f: N → R be defined as f(x) = 2x. Show that f is one-one but not onto.

8. Let f: R → R be defined by f(x) = x² + 4. Find f⁻¹(5) if it exists.

9. Show that the relation R in the set A = {1, 2, 3, 4, 5} given by R = {(a,b) : |a-b| is even} is an equivalence relation.

10. Let f: R → R be defined by f(x) = 3x - 7. Show that f is invertible and find f⁻¹.

11. Check whether the relation R in R defined by R = {(a,b) : a ≤ b³} is reflexive, symmetric or transitive.

12. Show that the function f: R → R given by f(x) = x³ + x is a bijection.

13. Let f: W → W be defined by f(n) = n - 1 if n is odd and f(n) = n + 1 if n is even. Show that f is bijective.

14. Consider f: R⁺ → [4, ∞) given by f(x) = x² + 4. Show that f is invertible and find f⁻¹.

15. Let * be a binary operation on Q defined by a*b = ab/4. Find the identity element and invertible elements.

16. Let A = N × N and * be the binary operation on A defined by (a,b)*(c,d) = (a+c, b+d). Show that * is commutative and associative.

17. Let f: R → R be defined by f(x) = (3 - x³)^(1/3). Prove that (fof)(x) = x.

18. Let * be a binary operation on the set Q of rational numbers defined by a*b = a + ab. Is * commutative? Is * associative?

19. Show that the relation R in the set Z of integers given by R = {(a,b) : 2 divides a - b} is an equivalence relation.

20. Let A = {1, 2, 3,...,9} and R be the relation in A × A defined by (a,b)R(c,d) if a + d = b + c. Show that R is an equivalence relation.

Conclusion

Relations and functions form the foundation of many advanced mathematical concepts. Understanding different types of functions, their compositions, and invertibility is crucial for higher mathematics. The concepts learned in this chapter will be extensively used in calculus, linear algebra, and other branches of mathematics.

Remember that practice is key to mastering these concepts. Work through the problems, draw diagrams to visualize functions and relations, and always verify whether a function is injective, surjective, or bijective before attempting to find its inverse.
